# Yousa-ddsp
[泠鸢yousa](https://space.bilibili.com/282994)的ddsp模型

---

## 初衷
因为学业压力，不能长时间玩泠鸢diffsinger声库，所以寻求一种轮椅的方法。

虚拟歌姬属于歌声合成类项目，ddsp属于声线转换类项目，理论上ddsp会更加轮椅一点。

---

## 模型特点

- 音域很广（ddsp的典型特征，以少许声音辨识度换取更好的高音表现）
- 具有一定的抗混响能力（不要太大，不然神仙难救）
- 遇到杂音会声音有点毛躁的感觉（因为我的推理源中的气声成分很多）
- 半御姐+半亮声线（清脆-->凝重）
- 抗底噪能力差，原因同上上（有时候会把噪声放大为气声处理）
